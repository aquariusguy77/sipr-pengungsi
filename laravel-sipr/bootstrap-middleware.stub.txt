Contoh registrasi middleware untuk proyek Laravel target

Jika proyek Laravel target memakai registrasi middleware terpusat, daftarkan middleware berikut:

- App\Http\Middleware\EnsureSiprAuthenticated
- App\Http\Middleware\EnsureSiprAbility

Contoh alias yang disarankan:
- sipr.auth => App\Http\Middleware\EnsureSiprAuthenticated
- sipr.ability => App\Http\Middleware\EnsureSiprAbility

Setelah alias tersedia, route bisa ditulis dengan pola:
- Route::middleware('sipr.auth')->group(...)
- ->middleware('sipr.ability:manage-refugees')
- ->middleware('sipr.ability:manage-documents')
- ->middleware('sipr.ability:review-changes')

Catatan:
- Middleware `EnsureSiprAuthenticated` menerima login Laravel asli atau sesi demo `sipr_user`.
- Paket ini sudah memakai class middleware langsung di routes web, jadi stub ini hanya bantuan untuk proyek yang ingin merapikan alias middleware.
