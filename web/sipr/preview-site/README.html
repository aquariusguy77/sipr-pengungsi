Paket Laravel SIPR

Panduan singkat menyalakan struktur Laravel saat environment PHP tersedia.

File ini merangkum apa yang sudah ada di paket `/workspace/output/laravel-sipr` dan langkah masuk akal untuk mengaktifkannya di project Laravel nyata.

TargetLaravel-ready

Route, controller, service, policy, seeder, factory, migration, dan view Blade sudah disiapkan.

Langkah Jalan

Urutan aktivasi

1. Pasang file ke project Laravel

Salin `app`, `resources/views`, `routes`, dan `database` ke project target.

2. Siapkan environment

Atur `.env`, koneksi database, dan user table yang memuat kolom `role`.

3. Jalankan migration dan seeder

Gunakan `php artisan migrate --seed` setelah semua dependensi siap.

4. Aktifkan policy dan auth

Pastikan `AuthServiceProvider` terbaca dan middleware auth digunakan pada route yang sesuai.

5. Jalankan server

Gunakan `php artisan serve` atau web server lokal yang biasa dipakai tim.

Yang Sudah Ada

Isi paket saat ini

Data dan struktur

Migration, seeder, factory, model relasi, dan service fallback ke sample data.

UI dan alur

Dashboard, data pengungsi, penempatan, dokumen, riwayat, laporan, pengaturan, dan wizard 4 langkah.

Kontrol akses

Policy stub untuk `Refugee`, `Placement`, dan `RefugeeDocument`, plus role flow dasar.
