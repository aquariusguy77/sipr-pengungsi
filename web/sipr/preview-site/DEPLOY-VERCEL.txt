Deploy Preview SIPR ke Vercel

Yang perlu di-upload
- seluruh isi folder `preview-site`

File opener
- `index.html`

URL penting setelah deploy
- `/` untuk Dashboard
- `/login` atau `/login.html` untuk halaman Masuk SIPR
- `/refugees` atau `/refugees.html`
- `/placements` atau `/placements.html`
- `/documents` atau `/documents.html`
- `/history` atau `/history.html`
- `/reports` atau `/reports.html`
- `/settings` atau `/settings.html`

Catatan
- Konfigurasi `vercel.json` sudah disiapkan agar URL bisa lebih rapi.
- Jika Vercel membaca project dari root folder, arahkan root deploy ke folder `preview-site`.
- Preview ini adalah static site untuk testing tampilan, bukan runtime Laravel.
