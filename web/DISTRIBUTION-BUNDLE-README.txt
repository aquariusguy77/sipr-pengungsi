SIPR Distribution Bundle

Isi bundle distribusi formal ini:
- `laravel-sipr-package-v15-production-cut.zip`
- `sipr-docs-config.zip`
- `EXPORT-MANIFEST.txt`
- `CHANGELOG-PACKAGES.txt`

Urutan pakai tercepat:
1. Baca `EXPORT-MANIFEST.txt`
2. Baca `CHANGELOG-PACKAGES.txt`
3. Pilih profil environment preview atau production dari `sipr-docs-config.zip`
4. Jika ingin merge penuh sekaligus, gunakan `laravel-sipr-package-v15-production-cut.zip`
5. Jika ingin merge bertahap, gunakan arsip pecahan sesuai manifest
